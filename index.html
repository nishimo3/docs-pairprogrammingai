Welcome to Pair Programming AI

"Pair Programming AI" is a partner who has knowledge of programming and can assist developers in software development. Pair programming AI can assist in software design, implementation, and verification, just as if an experienced software developer were standing next to you to help you. We are developing various prototype tools to realize Pair programming AI.
